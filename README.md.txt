OVERVIEW
------------------------------------------------------------
Dephaze describes a goal-directed, non-local, and self-regulating
field (Ψ) from which observable reality unfolds. It replaces the
idea of a fixed spacetime or random process with an adaptive
projection from a timeless ground state (Ω₀) toward a self-generated
target configuration (Imago).

The system’s coherence is defined by:
    Ξ = φ³ / φ⁻³

Ξ > 1  → Generative phase dominates (evolution)
Ξ ≈ 1  → Critical balance (maximal complexity)
Ξ < 1  → Pattern dominance (stasis)

Time is the sequence of projections (φ³ samplings).
Space is the topology of these projections (φ⁻³ pattern).
